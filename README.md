## 它到底在做什么？

Arrrt 是一个随身美术馆 App，iOS 和安卓都能用，主打每天推送一幅世界名画，配上中文原创的画作赏析文字。

这里头有几个细节值得说一下。

**内容是中文原创的。** 很多类似 App 的做法是把博物馆的英文介绍机翻成中文——那种中文读起来像是 Google 翻译加上学术腔，看了头疼。Arrrt 的团队自己写，还会跟博物馆的权威资料比对核实（比如他们说，不会说梵高割掉了"耳朵"，因为其实割的是"耳垂"，这个细节他们较真了）。

**画作是精选的，不是堆量。** 每天一幅，由编辑人工挑选。春天可能出现梵高画的杏花，中秋节推唐伯虎的明月——有点节气感，不是乱放的。

**界面做了减法。** 打开就是一幅画，挂在白墙上那种效果。边框颜色、图标颜色都会跟当天的画作匹配。不花哨，但整体感觉收拾得挺干净。

已经拿到了苹果、华为、OPPO、小米等主流应用商店的精选推荐，累计推荐次数超过五十几次，这个数字在国产独立 App 里算挺少见的。
